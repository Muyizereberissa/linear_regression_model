##  Public API Endpoint
🔗 https://temperature-prediction-j6aq.onrender.com/

Use the `/predict` endpoint to send input data and receive predicted temperature values.  
The API is testable using Swagger UI.

Use the `/retrain` endpoint to provide new data and target values to update the model.  
This allows the model to improve over time and adapt to new patterns.
